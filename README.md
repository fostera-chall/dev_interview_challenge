# 🧪 Java Debugging Exercise

## 🎯 Objective

The goal of this exercise is to evaluate your ability to:

* Understand an existing codebase
* Identify and fix bugs
* Write clean and maintainable code
* Improve code quality when relevant

---

## 🧱 Context

You are working on a simple order management application.

The application is responsible for:

* Calculating the total price of an order
* Applying discounts
* Handling basic customer information

Some incorrect behaviors have been reported, and your task is to investigate and fix them.

---

## 🚨 Your Mission

1. Identify issues in the codebase
2. Fix the bugs
3. Ensure the application behaves correctly
4. Add or update tests if necessary

---

## ▶️ Getting Started

You can use **IntelliJ IDEA** or any Java IDE of your choice.

---

## 📝 Bonus (Optional)

* Improve the code structure
* Enhance error handling
* Add comments or documentation where useful
* Suggest improvements (in this README or in comments)

---

## 📦 Submission

Please provide:

* A link to your repository (GitHub preferred)
* Or a zip archive of your project

---

## 💬 Tips

* Take time to understand the code before making changes
* Feel free to use logs or a debugger
* Keep things simple and pragmatic
* If something is unclear, state your assumptions

---

Good luck 🚀
